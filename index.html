Garden App

A community based app for growing, connecting, and learning. Show off what you're growing and what you have to say, connect with others, and explore what's growing in the neighborhood.

Technology

Linux, Nginx, PostgreSQL, Python stack. App is designed using the Flask framework for a Python powered, back-end heavy application.

Catalog App

A simple web app to manage content or inventory. Users can use third party authentication and authorization to sign in and explore and manage the content of the site.

Technology

The Catalog App utilizes Python's Flask framework to serve a responsive web app. On the backend the app runs on an Amazon Web Services Lightsail instance, a virtual Linux server and is configured to use an Nginx web server and a PostgreSQL database server. Python's WSGI HTTP Server, Gunicorn, is used to serve the Flask app.
